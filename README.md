# Non-Terrestrial network enabled Integrated Access and Backhaul feature for ns-3 mmwave module #
Integrated Access and Backhaul (IAB) is a 3GPP Study Item on the possibility of using the same network equipment and resources for both access and backhaul.

This is an extension of the [ns-3 mmWave module](https://github.com/nyuwireless-unipd/ns3-mmwave "ns-3 mmWave repo") combining Integrated Access and Backhaul feature from [ns3-mmwave-iab](https://github.com/nyuwireless-unipd/ns3-mmwave) and Non-terrestrial network [ns-3-ntn](https://gitlab.com/mattiasandri/ns-3-ntn). This framework has been developed by the University of Manitoba. It is still a _work in progress_.
